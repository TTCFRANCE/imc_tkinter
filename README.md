# Calculateur d'IMC (Python - Tkinter)

## Description
Ce projet est une application avec interface graphique permettant de calculer l'IMC (Indice de Masse Corporelle).  
L'utilisateur entre son poids et sa taille, puis le programme calcule et affiche son IMC.

## Objectif
J'ai réalisé ce projet pour apprendre :
- la création d'interfaces graphiques avec Tkinter
- la gestion des événements (clic sur un bouton)
- la récupération et le traitement des données utilisateur
- la gestion des erreurs

## Fonctionnalités
- Saisie du poids (kg)
- Saisie de la taille (cm)
- Calcul automatique de l'IMC
- Affichage du résultat
- Gestion des erreurs si les données sont invalides

## Fonctionnement
1. L'utilisateur ouvre la fenêtre IMC
2. Il entre son poids et sa taille
3. Il clique sur "Calculer"
4. L'IMC est calculé avec la formule :

IMC = poids / (taille²)

## Technologies utilisées
- Python
- Tkinter
